# Sistem-Monitoring-Cargo
Sistem ini merupakan Smart Warehouse Monitoring System Kabuto Cargo berbasis bahasa C. Sistem mencatat data lima barang, menentukan status stok aman, waspada, atau kritis, menampilkan tabel terstruktur, visual bar stok, rekomendasi restock, skor kesehatan gudang, serta menu interaktif berulang yang mendukung pengambilan keputusan logistik.

- dari soal latihan di modul UAP PEMOGRAMAN DASAR SEMESTER 1
